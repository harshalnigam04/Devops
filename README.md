# Python Practice Programs

This repository contains four basic Python programs that demonstrate fundamental concepts such as conditionals, dictionaries, and file handling.

---

# 1. Grade Checker
Description
This program takes a student's score as input and prints the corresponding grade based on the following criteria:

- 90 and above → A  
- 80–89 → B  
- 70–79 → C  
- 60–69 → D  
- Below 60 → F  

# Concepts Used
- if-elif-else statements  
- User input handling  


# 2. Student Grades Manager

# Description
This program uses a dictionary to store student names and their grades. It allows the user to:

- Add a new student and their grade  
- Update an existing student’s grade  
- Display all student grades  

# Concepts Used
- Dictionaries  
- Basic operations (add, update, display)  
- Conditional statements (if-else)  



# 3. Write to a File

# Description
This program creates a text file and writes content into it.

# Concepts Used
- File handling in Python  
- open() function in write mode ('w')  
- write() method  

# 4. Read from a File

# Description
This program reads content from a text file and displays it on the screen.

# Concepts Used
- File handling in Python  
- open() function in read mode ('r')  
- read() method  


# Summary

These programs are designed for beginners to practice:
- Conditional logic  
- Data structures (dictionary)  
- File input/output operations  

They provide a strong foundation for understanding core Python programming concepts.
